Một phút là chụp được

Tải & mở file .dmg
Kéo FC-FastCapture vào thư mục Applications.
Mở lần đầu: chuột phải → Open
App chưa ký Apple nên lần đầu cần chuột phải → Open → xác nhận.
Cấp 2 quyền → khởi động lại
Bật Screen Recording + Accessibility trong cửa sổ "Cấp quyền" → bấm khởi động lại. Xong.
Vì sao có 2 quyền? Screen Recording để chụp ra nội dung thật (không phải hình nền); Accessibility để phím tắt toàn cục chạy. App không thu thập, không gửi dữ liệu đi đâu — quyền chỉ dùng tại chỗ.
Trên Windows còn nhanh hơn: tải FC-FastCapture-Windows.exe → chạy thẳng, không cần cấp quyền. Lần đầu nếu hiện SmartScreen thì bấm More info → Run anyway. Phím tắt mặc định là Ctrl+Alt+1…5 (đổi được trong Settings).

macOS .dmg ~58–68MB · Windows .exe ~89MB · không quảng cáo, không thu phí, không giới hạn
Máy Mac không rõ đời? Bấm Apple → About This Mac: thấy Apple M1/M2/M3/M4 → chọn Apple Silicon; thấy Intel → chọn Mac Intel. Máy Windows → tải file .exe.
Windows lần đầu mở: có thể hiện SmartScreen → bấm More info → Run anyway (app chưa mua chứng chỉ Microsoft — như bản Mac cần chuột phải → Open).
